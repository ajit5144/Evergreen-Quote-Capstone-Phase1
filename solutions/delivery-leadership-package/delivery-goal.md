# Delivery Goal — Evergreen Quote

## Goal
Create and deliver the Evergreen Quote website with all planned MVP functionality, and complete the pull request process to merge the solution into main, ensuring the application is fully ready for stakeholder demonstration.

## "Done" looks like

- The application is deployed to main with Evergreen branding, styling, and responsive design across mobile, tablet, and desktop devices.
- The premium calculation functionality successfully returns a premium amount based on the defined calculation logic.
- The CI pipeline executed against the merge commit completes successfully with a green status.
- A single reviewed and approved pull request is merged into main, and the feature branch is deleted after merge.

## Out of scope (this week)

- Security Vulnerabilities will be fixed with later build
- Website will show a sample premium. Actual premium might vary. Final data will be updated before Prod run
- Text and Email functionality will not be available as part of MVP
- Customer Policy, Account creation as well as Proof of Insurance generation will not be part of this MVP
- System will not accept Claims and Billing related details. This has to be built later
- System will not store any customer related information.
